# ArtisanHub — Marketplace de Artesanato Brasileiro

Plataforma B2C para valorização e comercialização de artesanato brasileiro, conectando mestres artesãos a consumidores que buscam produtos autênticos e culturalmente ricos.

## Tecnologias

- **React 19 + TypeScript** — interface responsiva com componentes reutilizáveis
- **Tailwind CSS v4** — design system com tokens customizados
- **Motion (Framer Motion)** — animações e micro-interações
- **Node.js + Express** — API REST backend
- **Vite** — build e servidor de desenvolvimento

## Como executar

**Pré-requisitos:** Node.js instalado

```bash
# Instalar dependências
npm install

# Iniciar em modo desenvolvimento
npm run dev
```

Acesse: `http://localhost:3000`

## Funcionalidades

- Marketplace com produtos em destaque e curadoria
- Catálogo com filtros por categoria e busca em tempo real
- Carrinho de compras funcional com drawer lateral
- Página de detalhe do produto com itens relacionados
- Painel do Artesão: submissão de peças para curadoria
- Painel Administrativo: aprovação/rejeição de produtos
- Páginas institucionais: Transparência, CMMI, IHC, Privacidade
- Login diferenciado para artesão e administrador
